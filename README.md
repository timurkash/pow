# "Fortune" TCP-server with protection from DDOS based on Proof of Work

## 1. Description
This project is a solution for some interview question on Golang.

## 2. Prerequisites
Install fortune
```bash
sudo apt-get install fortunes fortune-mod
```

## 3. Run
```bash
make start
```
